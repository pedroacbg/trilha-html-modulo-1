O que são Clients?

Índice

Em um âmbito geral, cliente é todo aquele que consome algo e para o contexto da internet não é diferente, os clientes são as máquinas e dispositivos que consomem a internet.
Todo dispositivo que está conectado a uma rede, que faz uma requisição e recebe uma resposta.
Para exemplificar, vamos observar o que acontece quando navegamos no google: Abrimos nosso navegador, digitamos o que estamos procurando e pressionamos enter, neste momento nossa máquina Client ira fazer uma requisição para um server, que ira retornar a página de busca.

O que são Servers?

Índice

Os Servers são os locais remotos onde ficaram "salvos" os arquivos da nossa aplicação. Quando um usuário solicita acesso a nossa aplicação Web, existe uma comunicação entre o Client-side e Server-side para processar as informações ou ações do usuário. Em resumo, o Server é onde podemos salvar nossa aplicação Web, banco de dados, servidores de e-mail e outros mais.

Como é uma estrutura básica de um HTML?

Índice

         <html lang="pt-br">

            <meta charset="UTF-8">
            <meta http-equiv="X-UA-Compatible" content="IE=edge">
            <meta name="viewport" content="width=device-width, initial-scale=1.0">
            <title>Document</title>
         </head>
         <body>

<!DOCTYPE html>
 Está tag é repsonsável por dizer ao navegador que é um documento do tipo HTML.

<html lang="pt-br">
 Está é uma tag que marca todo o conteudo html, se observar bem, verá que ela começa com html e termina com /html, desta forma marcando todo conteúdo.

<head>
 Está tag é responsável por abraçar toda a cabeça do site, aqui entram alguns metadados como o titulo da página, folhas de estilos e alguns outros arquivos e configurações.

<meta charset="UTF-8">
 Está é uma tag de configuração, aqui dizemos ao navegador qual é o formato de codificação de caracteres utilizado.

<meta http-equiv="X-UA-Compatible" content="IE=edge">
 Tag de compatibilidade com internet explorer

<meta name="viewport" content="width=device-width, initial-scale=1.0">
 Está tag é responsavel por controlar a exibição do nosso site em dispositivos móveis.

<title>Document </title>
 Esta tag é utilizada para marcar o titlo do site, o que aparece na aba do navegador.

</head>
 Tag de fechamento da cabeça do nosso html.

<body>
 Está é tag que irá exibir nosso conteúdo, ela abraça nosos paragrafos imagens e todo tipo de midia.

</body>
 tag de fechamento do corpo do conteudo.

</html>
 tag de fechamento do documento html.

Entendendo a diferença entre listas ordenadas e não ordenadas

Índice

As listas não numeradas são usadas para listar itens, sem se preocupar com sua sequência. Chamamos de lista de marcadores apenas.

As tags usadas para criar uma lista não ordenada são <ul> e <li>. A tag <ul> é usada para definir a lista e a tag <li> é usada para cada item da lista.


As listas ordenadas ou numeradas são usadas para indicar alguma sequência ou numeração

As tags usadas para criar uma lista não ordenada são <ol> e <li>. A tag <ol> é usada para definir a lista e a tag <li> é usada para cada item da lista.
